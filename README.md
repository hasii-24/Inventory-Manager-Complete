📦 Inventory Management System

📌 Project Overview

This project is a full-stack Inventory Management System designed to manage product stock, user purchases, and administrative controls.

The system provides separate dashboards for Admin and Users, ensuring proper access control and functionality.

🚀 Features

👨‍💼 Admin Functionalities:

View all available products
Add new stock
Update existing product details
Track purchased products

👤 User Functionalities:
View available products with quantity
Purchase products
View order details on homepage/dashboard

🛠️ Tech Stack

Frontend:
HTML
CSS
Bootstrap
JavaScript

Backend:
Node.js
Express.js

Database:
MongoDB (Atlas)

🧩 System Architecture

The application follows a client-server architecture:

Frontend interacts with backend APIs
Backend handles business logic
Database stores product and user data

Implemented role-based access (Admin/User)
Worked with MongoDB Atlas
